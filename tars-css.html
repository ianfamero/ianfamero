Tars CSS

Note: This is no longer maintained. Just keeping it here for legacy purposes.

Tars CSS is a collection of my most commonly used CSS classes and styles.

It keeps me from having to write code from scratch for every new project, saving me a lot of valuable time and effort. There are a lot of CSS frameworks out there that are designed for this purpose, but most of them are cluttered with too much predefined styles that I wouldn't ever use.

That's why I decided to write my own.

Tars CSS is written in SASS which gives me the ability to use variables and mix-ins, among other things that SASS has to offer. It has a 12-column responsive grid system that I wrote myself, as well as some predefined classes for styling tables, lists, form elements, and more.

I have made a lot of changes with it since I first wrote it n years ago. It’s still far from perfect, but it works for me, and it helps me produce better results in a much shorter period of time.

This project is available on Github.
