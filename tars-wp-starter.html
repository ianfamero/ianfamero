Tars WP Starter

Note: This is no longer maintained. Just keeping it here for legacy purposes.

Tars WP Starter is a WordPress starter theme built on top of Tars CSS.

It contains all the necessary files and folders for a custom WordPress theme to work. One caution of advice for whoever plans to use it is that it is not meant to be used as a parent theme. It was only meant to be used as a baseline theme for creating custom WP themes.

Since it comes bundled with Tars CSS, you wouldn't really have to write a lot of code. Just call the predefined classes and styles on Tars CSS and you're good to go!

This project is available on Github.
